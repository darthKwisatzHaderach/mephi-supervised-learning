#### **–°—Ä–∞–≤–Ω–µ–Ω–∏–µ: —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π vs –û–û–ü —Å—Ç–∏–ª—å ‚Äî –ø—Ä–∏–º–µ—Ä —Å –¥–≤—É–º—è –ø–æ–¥–≥—Ä–∞—Ñ–∏–∫–∞–º–∏**

##### üîπ **1. –§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π —Å—Ç–∏–ª—å**
```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 20)
y = 2 * x + 3 + np.random.normal(0, 2, size=x.shape)

plt.figure(figsize=(10, 4))
plt.subplot(1, 2, 1)
plt.scatter(x, y)
plt.title('–õ–µ–≤—ã–π –≥—Ä–∞—Ñ–∏–∫')

plt.subplot(1, 2, 2)
plt.scatter(x, y)
plt.title('–ü—Ä–∞–≤—ã–π –≥—Ä–∞—Ñ–∏–∫')

plt.suptitle('–§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π —Å—Ç–∏–ª—å')
plt.tight_layout()
plt.show()
```

##### üîπ **2. –û–û–ü —Å—Ç–∏–ª—å**
```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 20)
y = 2 * x + 3 + np.random.normal(0, 2, size=x.shape)

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

axes[0].scatter(x, y)
axes[0].set_title('–õ–µ–≤—ã–π –≥—Ä–∞—Ñ–∏–∫')

axes[1].scatter(x, y)
axes[1].set_title('–ü—Ä–∞–≤—ã–π –≥—Ä–∞—Ñ–∏–∫')

fig.suptitle('–û–û–ü —Å—Ç–∏–ª—å')
fig.tight_layout()
plt.show()
```

##### üîπ **–ö–ª—é—á–µ–≤—ã–µ —Ä–∞–∑–ª–∏—á–∏—è (–≤ —Ç–∞–±–ª–∏—Ü–µ –±–µ–∑ –∫–æ–¥–∞):**

| –ê—Å–ø–µ–∫—Ç | –§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π —Å—Ç–∏–ª—å | –û–û–ü —Å—Ç–∏–ª—å |
|-------|------------------------|-----------|
| **–°–æ–∑–¥–∞–Ω–∏–µ –≥—Ä–∞—Ñ–∏–∫–∞** | `plt.figure()` | `fig, axes = plt.subplots()` |
| **–ê–∫—Ç–∏–≤–∞—Ü–∏—è –ø–æ–¥–≥—Ä–∞—Ñ–∏–∫–∞** | `plt.subplot(1, 2, 1)` | –†–∞–±–æ—Ç–∞ –Ω–∞–ø—Ä—è–º—É—é —Å `axes[0]`, `axes[1]` |
| **–ù–∞—Å—Ç—Ä–æ–π–∫–∞ –∑–∞–≥–æ–ª–æ–≤–∫–∞** | `plt.title()` | `axes[0].set_title()` |
| **–û–±—â–∏–π –∑–∞–≥–æ–ª–æ–≤–æ–∫** | `plt.suptitle()` | `fig.suptitle()` |
| **–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å** | –ë—ã—Å—Ç—Ä–∞—è –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è | –°–ª–æ–∂–Ω—ã–µ –≥—Ä–∞—Ñ–∏–∫–∏, –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ, –ø–æ–≤—Ç–æ—Ä–Ω–æ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ |
